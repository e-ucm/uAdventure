This sample demonstrates how to install a TrueType font on the target system.Execute corresponding .cmd file to build desired msi. Then execute the .msi to start the installation.-------------------The font 'Free Sans Bold' (FreeSansBold.ttf) is not used by Wix# in any way and it is only provided as a sample. The font is obtained from https://savannah.gnu.org/projects/freefont/ where it's licensed under 'GNU General Public License v3 or later'
